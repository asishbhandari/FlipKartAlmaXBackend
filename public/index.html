Api's for Flipkart Backend

This is a simple api application offering various endpoints to handle tasks efficiently. Leveraging NodeJs with ExpressJs, it provides a scalable and easy-to-use framework for building RESTful APIs. The application's provides different endpoints regarding user, products, categories, cart and Wishlist. In addition, it gracefully handles undefined routes by responding with a 404 status code and an appropriate error message.
